# Acceptance Criteria

以下を満たしたら初期版完成とみなす。

## 必須

- 4人対戦で 1 ゲーム最後まで進行できる
- 人間2人 + CPU2人でオンライン対戦できる
- ルーム作成と参加ができる
- CPU難易度が 3 段階ある
- 設定画面でローカルルールを ON/OFF できる
- 合法手以外を出せない
- サーバ側で手の正当性を再検証する
- 順位が 1 位〜4 位まで確定する
- タイトル画面、ルーム作成/参加画面、設定画面、対戦画面、結果画面がある
- 7渡し、10捨て、12ボンバー、カード交換で追加選択 UI が動作する
- 設定に応じて以下が動く
  - 階段
  - 革命
  - 8切り
  - スペ3返し
  - 縛り
  - 5スキップ
  - 7渡し
  - 10捨て
  - 11バック
  - 12ボンバー
  - カード交換

## 望ましい

- スマホでも遊べる
- 再接続に最低限対応する
- CPU の行動が不自然すぎない
- 主要ロジックにテストがある
- UI とゲームロジックが分離されている
