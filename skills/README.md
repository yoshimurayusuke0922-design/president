# skills フォルダ

このフォルダは、必要になった reusable な補助資産を置くための場所です。

## 想定用途

- セットアップ手順
- コード生成テンプレート
- デバッグ補助
- プロンプト断片
- 補助スクリプト
- 変換ツール

## 例

- `skills/setup/dev-env.md`
- `skills/debug/socket-checklist.md`
- `skills/codegen/state-machine-template.md`
- `skills/prompts/local-rule-extension.md`

## 置かないもの

- 本番アプリ本体
- 画面コード
- ルーム管理ロジック
- 一時メモ
